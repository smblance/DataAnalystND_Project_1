Analysis of the Stroop test performance.¶

by Mark Ayzenshtadt

0. The data.¶

The data consists of the performance in the Stroop test for 24 participants.
Each row shows result in seconds in the congruent and incongruent tasks for one participant.

1. The independent and dependent variables.¶

Independent variable: Congruency (whether the color of the word matches the word).
Dependent variable: Performance (measured by the time it takes to name the colors of all the words).

2. Set of hypotheses for the task.¶

It is important to notice that the samples are dependent (paired) - both tasks are done by the same person.
To test whether congruency matters, we will test whether the samples come from the same distribution. So, the null hypothesis is: "The two samples come from the same distributions."
The alternative hypothesis: "The two samples come from different distributions."

To choose the test that we want to use, we will test the two samples for normality. We will do it with the scipy.stats' "Normaltest", which is based on the analysis of skew and kurtosis of the distribution, and also with the Shapiro-Wilk test.
If both samples will pass the normality test, we will use the paired Student's T-test, and if either sample does not pass - the Wilcoxon signed-rank test.

We see that both normality tests produce the same results: Congruent data is normal, and Incongruent is not.
So, instead of the Student's T-test we will use one-sided Wilcoxon signed-rank test. We use one-sided test because every participant did worse in the Incongruent task.

print 'Wilcoxon signed-rank test:\n', stats.wilcoxon(x = data.Congruent, y = data.Incongruent)

Wilcoxon signed-rank test:
WilcoxonResult(statistic=0.0, pvalue=1.821529714896801e-05)

The (one-sided) critical value at 1% probability for the sample size of 24 is 69.
We will reject the null hypothesis if test statistic is less than 69.
The statistic that we obtained is 0.0, and therefore we reject the null hypothesis. The p-value that we obtained is about 0.000018.

6. What is responsible for the effects observed? What alternative or similar tasks would result in similar effect?¶

The main theory behind the effect is that the two streams of information, the color of the text and the meaning of the word, are processed in parallel, and the meaning is processed faster. When the meaning is processed, the brain is ready for response selection, and it takes time and effort to:

  • understand that the response is wrong,
  • wait for the color to be processed,
  • finally choose the correct response.

We are used to a lot of reading, so word processing is done much more automatically and needs less attention than color processing.
If we change the task, asking the subject to report the information that is processed faster (word meaning), than the lag before the answer is much smaller, because the response selection can be done before the conflicting information (color) is even processed.

Some of the stimuli that we can use to design similar tests are:

  • color of the word
  • meaning of word (can be colors/numbers/shapes/etc.)
  • emotion of the word
  • shape
  • size of the word or individual letters
  • sound of the word or sound of some animal or natural phenomenon, e.g. rain
  • number of the words or objects, e.g."55555"
  • location of the object inside a square
  • touch ("hard", "soft", "hot", "cold", "long", "short", etc)

We can make the participant experience incongruency with almost any combination of these stimuli.

So, the following tests are expected to produce effects similar to the Stroop test:

  • name the length of the given string of numbers. Correct answers will be: "555" -> 3, "22222" -> 5
  • name the location of the word inside the square. If the word "left" is at the bottom side of the square, the subject must report "bottom".
  • name the number with the biggest size. If "2" is written in the larger font than "5", the subject must report "2".
